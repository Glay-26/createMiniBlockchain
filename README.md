mini blockchain's main functions:
迷你区块链主要实现了以下功能：
    help [command...]    Provides help for a given command.
    exit                 Exits application.
    trans <to> <amount>  转账
    detail <index>       查看区块详情
    mine                 出块奖励
    chain                查看区块链
    pub                  查看本地地址
    peers                查看网络节点列表
    chat <message>       跟别的节点hi一下
    balance <address>    查看余额
    pending              查看还没有被打包的交易
